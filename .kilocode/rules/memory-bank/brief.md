Create a modern web optimized tower defense game based on game called "Server Survival", found here: https://github.com/stansz/server-survival. This game needs to perform well on both mobile and desktop Web browsers with PWA support.

**Project Status**: Core game systems are complete. Ready to implement game scenes that integrate all systems together.

**Key Features**:
- Educational tower defense game teaching cloud architecture concepts
- 6 defensive services (Firewall, CDN, Load Balancer, Cache, Database, Auto Scaler)
- 6 traffic types (Static, Read, Write, Upload, Search, Malicious)
- Resource management (Budget, Reputation, Service Health)
- Wave progression with escalating difficulty
- Random events (traffic surge, DDoS, degradation, budget bonus)
- Two game modes: Survival and Sandbox
